# 🏥 MedTalk – AI Medical Chatbot

MedTalk is a simple and beginner-friendly AI-powered medical chatbot built using Flask and Google Gemini API. It provides short, easy-to-understand health information along with basic medicine suggestions and a safety disclaimer.

---

## 🚀 Features

- 💬 AI-powered chatbot using Gemini API
- 🧠 Simple and beginner-friendly medical responses
- 🎤 Voice input support (Speech Recognition)
- ⚡ Fast and lightweight Flask backend
- 📱 Clean and responsive UI
- ⚠️ Includes medical disclaimer for safety

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python (Flask)
- **AI API:** Google Gemini API

---

## 🎤 Voice Feature

- Click the 🎙️ button to start speaking
- Your voice will be converted to text automatically
- The chatbot responds instantly

---

## ⚠️ Disclaimer

This chatbot provides general health information only.  
Always consult a qualified doctor before taking any medication or making medical decisions.
